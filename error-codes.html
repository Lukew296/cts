Error Codes Documentation

A comprehensive list of potential site error codes and their meanings.

ERR-404: Page Not Found

The requested resource could not be found on the server. This usually means the URL is incorrect or the page has been moved/deleted.

ERR-500: Internal Server Error

An unexpected condition was encountered by the server. This is a general error message indicating something went wrong on the server side.

ERR-401: Unauthorized

Authentication is required and has failed or has not yet been provided. You might need to log in.

ERR-403: Forbidden

The server understood the request but refuses to authorize it. You do not have permission to access this resource.
